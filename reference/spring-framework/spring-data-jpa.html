Spring Data JPA

Maven Dependency

<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-data-jpa</artifactId>
</dependency>

Database Entity

In domain package you can define a Database Entity:

@Entity (1)
public class Author {

    @Id (2)
    @GeneratedValue(strategy = GenerationType.AUTO) (3)
    private Long id;

    private String firstName;

    private String lastName;

    // Getters and Setters
    ...

}
1 @Entity specifies that the class is a Database Entity
2 @Id specifies the primary key of an Entity
3 @GeneratedValue specifies the generation strategy for the value of primary key. See üü£ GenerationType for generation strategies
