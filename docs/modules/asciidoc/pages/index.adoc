====

include::ROOT:partial$java-type-legend.adoc[lines=2..-1]
====
[,adoc]
----

.Include Content by Tagged Region
[%collapsible]
====
Include Tagged Region:

include::index.adoc[tag=example-tag,indent=0]
====
You can tag a region:
[,adoc]
----
// tag::example-tag[]
This is Tagged Region
// end::example-tag[]
----
And then include it:
[,adoc]
----
Include Tagged Region:

\include::index.adoc[tag=example-tag,indent=0]
